# Terminal Commands

`npm init -y` - initializes npm and creates package.json

`npm i typescript @types/node --save-dev` - dependencies for TS dev

`tsc` - transpile ts into js

`node dist/${file-name}` - execute specified js file in dist folder

`npm i accounting` - number and currency formatting libaray (javascript) package for money import

`npm i nodemon -g` - global packapge for auto-restarts on save change
